Bloques (Grid de tarjetas)

Flexbox

Útil para alinear elementos en una dimensión: filas o columnas (nav, botones, etc.).

display:flex gap wrap

Grid

Ideal para layouts en dos dimensiones: filas y columnas (tarjetas, galerías, dashboards).

display:grid repeat() fr

Hover

Microinteracciones: feedback visual al usuario (botones y tarjetas).

:hover transition transform

Responsive

Adaptación por puntos de ruptura: móvil (1 col), tablet (2 col), escritorio (3 col).

@media breakpoints mobile-first

Accesibilidad

Semántica, contrastes y reducir animación si el usuario lo solicita.

aria-label semántica prefers-reduced-motion

Buenas prácticas

Variables CSS, contenedor central, tamaños fluidos y estilos reutilizables.

:root clamp() componentes

Práctica guiada (10 min)

  1. Cambia el breakpoint de móvil de 599px a 640px y observa el efecto.
  2. Modifica la rejilla de escritorio a 4 columnas: repeat(4, 1fr).
  3. Añade una nueva tarjeta y comprueba cómo se recoloca automáticamente.
  4. Personaliza el hover (por ejemplo, aumentando la elevación o cambiando el borde).
